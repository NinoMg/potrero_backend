# Sistema de Gestión de Productos (PHP + MySQL)

Aplicación web desarrollada en PHP para la gestión de productos con funcionalidades CRUD completas.

## Funcionalidades

- Login de usuario
- Alta, edición y eliminación de productos
- Gestión de stock
- Manejo de imágenes
- Persistencia en base de datos MySQL

## Tecnologías

- PHP
- MySQL
- Bootstrap
- HTML / CSS

## Descripción

El sistema permite administrar productos mediante un panel de control, incluyendo la carga, edición y eliminación de registros con almacenamiento en base de datos.

## Instalación

1. Clonar repositorio
2. Configurar base de datos en MySQL
3. Ejecutar en servidor local (XAMPP / WAMP)
